# Quick Start Guide

## Setup (One-Time)

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Preprocess Data (REQUIRED - First Time Only)
```bash
python preprocess_data.py
```

**What this does:**
- Converts dates to Hijri calendar
- Identifies Ramadan periods
- Detects Islamic events
- Translates Arabic to English
- Takes 5-15 minutes

**Wait for this message:**
```
âœ“ SUCCESS: Data preprocessing completed successfully!
```

### Step 3: Run Dashboard
```bash
python dashboard_app.py
```

**Open browser to:** http://localhost:8050
